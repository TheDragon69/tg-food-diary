# Дневник Питания с ИИ — Telegram Mini App

Легковесное Telegram Mini App с серверной частью, которое позволяет пользователям вести дневник питания, просто делая фото. Работает на базе API компьютерного зрения Logmeal для распознавания еды.

## Особенности
- **Распознавание с помощью ИИ**: Использует API Logmeal для мгновенного определения блюд и их точного КБЖУ (калории, белки, жиры, углеводы).
- **Автономный интерфейс (Offline First)**: Сохраняет историю локально внутри Telegram с использованием `localStorage`.
- **Удаление свайпом**: Нативный опыт управления приемами пищи, как в мобильных приложениях.
- **Контроль порций**: Легкое изменение размера порции с помощью ползунка-множителя.
- **Динамическая тема**: Автоматически подстраивается под светлую/темную тему пользователя через нативные переменные Telegram SDK.

## Технологический стек
- **Frontend**: Vanilla JavaScript, HTML5, CSS3, Telegram WebApp SDK.
- **Backend**: Node.js, Express.js.
- **ИИ-провайдер**: API Logmeal (Сегментация -> Пищевая ценность).

## Инструкции по установке

### 1. Получение API-ключа
1. Перейдите на [Logmeal.es](https://logmeal.es/api/) и создайте бесплатный аккаунт.
2. Перейдите в панель управления и сгенерируйте свой API-ключ.

### 2. Локальная разработка
1. Перейдите в папку `/backend`.
2. Выполните команду `npm install`.
3. Переименуйте файл `.env.example` в `.env` и вставьте ваш API-ключ Logmeal.
4. Выполните команду `npm start`. (Сервер запустится на порту 3000).
5. Откройте файл `/frontend/index.html` в браузере для тестирования интерфейса (включите режим мобильного просмотра в инструментах разработчика).

### 3. Развертывание на Railway (бесплатный тариф)
1. Запушьте этот репозиторий на GitHub.
2. Перейдите на [Railway.app](https://railway.app/), нажмите "New Project" -> "Deploy from GitHub repo".
3. Выберите этот репозиторий. Если вы деплоите только бэкенд, убедитесь, что Railway использует `package.json` из папки `/backend` для запуска Express-сервера.
4. Перейдите на вкладку **Variables** в Railway и добавьте переменную `LOGMEAL_API_KEY`.
5. После развертывания Railway предоставит вам публичный URL (например, `https://food-diary-production.up.railway.app`).
6. Обновите переменную `BACKEND_URL` в файле `frontend/app.js` (строка 6), указав ваш новый URL от Railway.

### 4. Подключение к Telegram
1. Разместите файлы из папки `/frontend` на любом бесплатном хостинге статических сайтов (Vercel, GitHub Pages, Netlify).
2. Откройте Telegram и напишите боту **@BotFather**.
3. Создайте нового бота с помощью команды `/newbot`.
4. Настройте кнопку меню Web App для вашего бота с помощью команды `/setmenubutton`.
5. Введите публичный URL вашего статического frontend-хостинга. 
6. Откройте вашего бота на телефоне и нажмите кнопку меню!
